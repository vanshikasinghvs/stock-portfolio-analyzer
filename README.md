# Stock Portfolio Analyzer

A Python project that analyzes historical S&P 500 stock performance.

## What it does
- Analyzes 9 major stocks including AAPL, MSFT, NVDA, AMZN, GOOGL, JNJ, JPM, META, and PG 
- Calculates annualized returns, volatility and Sharpe ratio
- Builds cumulative return charts to visualize performance over time
- Portfolio achieved 28.38% annualized return with a Sharpe ratio of 1.03

## Tools used
- Python
- Pandas
- NumPy
- Matplotlib
